Description

Here is where all RestClient Requests Arrive.

Planning to use spring boot and have anotations on each of these methods to accept HTTP/HTTPS Requests.

Operations

Visibility Name Description
public authenticate()

All User Authentications are sent to this method. It will Authenticate against existing users in MongoDB. DataBase and Collection is to be decided on a later date.
public get()

All Document Request come here. To fulfill a request we must first.

  1. Run SecurityValidator to see if the Request can be fulfilled.
  2. Get Document from MongoDB
  3. Filter Out Parameters that requesting user has no right to access.
public put()

All Document Write New Request come here. To fulfill a request we must first.

  1. Run SecurityValidator to see if the Request can be fulfilled.
  2. Filter Out Parameters that requesting user has no right to put.
  3. Check Parameters against Parameter format defined by Developer.
public update()

All Document Update Request come here. To fulfill a request we must first.

  1. Run SecurityValidator to see if the Request can be fulfilled.
  2. Filter Out Parameters that requesting user has no right to update.
  3. Check Parameters against Parameter format defined by Developer.
public delete()

All Document Delete Request come here. To fulfill a request we must first.

  1. Run SecurityValidator to see if the Request can be fulfilled.
  2. Delete Document. Or Fail to do so.
